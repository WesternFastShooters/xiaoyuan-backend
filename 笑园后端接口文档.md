## token鉴权

| 身份   | 权限                                                         | 违反权限的返回                                           |
| ------ | ------------------------------------------------------------ | -------------------------------------------------------- |
| 游客   | 可以访问  [/display/**](/display/**)                         | msg: {<br/> err_code:17,<br/> err_content:"无权限"<br/>} |
| 用户   | 可以访问  [/user/**](/user/**  )   或  [/display/**](/display/**) | msg: {<br/> err_code:17,<br/> err_content:"无权限"<br/>} |
| 管理员 | 可以访问   [/display/**](/display/**)   或   [/administrator/**](/administrator/**) | msg: {<br/> err_code:17,<br/> err_content:"无权限"<br/>} |

**其他异常**

```JavaScript
msg: {
  err_code:19,
  err_content:"无法验证访问权限"
}
```

```javascript
msg: {
 err_code:20,
 err_content:"token解析错误"
}
```

### CooperationUnit

- **cooperationUnitID    合作单位ID  string**

  - 生成ID代码如下

  - ```javascript
    () => {
            let d = new Date().getTime();
            let cooperationUnitID = 'xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'.replace(/[xy]/g, function (c) {
                let r = (d + Math.random() * 16) % 16 | 0;
                d = Math.floor(d / 16);
                return (c == 'x' ? r : (r & 0x3 | 0x8)).toString(16);
            });
            return cooperationUnitID;
        }
